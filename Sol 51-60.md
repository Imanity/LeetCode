# Question 55 	Jump Game

### Question Describe

给定一整数列, 其每个位置数字代表从该位置开始起跳能到达的最远距离, 判断从开始位置起跳能否到达末尾

### My Solution (AC)

对每一位置n, 找到其之前能到达的最远位置xn, 从 n - 1 到 1 依次判断 i + a[i] 是否大于等于 n, 若满足则 xn = i, 若无法找到满足的i则返回false, xn = 1时返回true, 时间复杂度o(n)
