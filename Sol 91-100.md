# Question 91   Decode Ways

### Question Describe

给定一串数字, 给定一种编码方式: 1 - 'A', ..., 26 - 'Z', 求该串数字可以是多少种字符串的编码

### My Solution (AC)

动态规划实现, 使用 d[i] 保存 s[0..i] 的可编码个数

- s[i + 1] == '0', d[i + 1] += d[i]

- s[i]和s[i + 1]可编码'K'到'Z'中一个, d[i + 1] += d[i - 1]

所求即为 d[s.size - 1]
